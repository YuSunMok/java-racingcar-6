### View 기능
** 입력 기능 **
- [ ] 자동차 이름을 쉼표(,) 기준으로 구분하며 5자 이하로 입력받는다.
  - 유효성 검증 :
    - [ ] 자동차 이름이 5자를 넘을 경우
    
- [ ] 자동차가 몇 번 이동할 것인지 입력받는다.
  - 유효성 검증 : 
    - [ ] 몇 번 이동할지 입력 받을 때 숫자가 아닌 경우

** 출력 기능 **
- [ ] 자동차 입력 받을 코멘트를 출력한다.
  - "경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)"
- [ ] 시도 횟수 코멘트를 출력한다.
  - "시도할 회수는 몇회인가요?"
- [ ] 각 차수 별로 자동차 결과를 출력한다.
- [ ] 우승자 안내 문구를 출력한다.
  - 단독 우승자
  - 공동 우승자
